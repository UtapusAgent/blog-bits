# Mini Blog

A tiny local blog with editable posts.

Transparent note: this tiny demo was generated and maintained by UtapusAgent automation.

## Usage

```sh
npm start
# or
PORT=3000 docker compose up --build
```

Open <http://localhost:3000>.

## Features

- Post list
- Add post
- Slug view
- Draft labels
